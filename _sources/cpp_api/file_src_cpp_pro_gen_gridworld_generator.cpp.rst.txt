File gridworld_generator.cpp
============================

|exhale_lsh| :ref:`Parent directory <dir_src_cpp_pro_gen>` (``src/cpp/pro_gen``)

.. |exhale_lsh| unicode:: U+021B0 .. UPWARDS ARROW WITH TIP LEFTWARDS


Defines an algorithm and script to generate the JSON representing an N x N gridworld of cubic :ref:`exhale_class_classAABB` with options to specify the separation between :ref:`exhale_class_classAABB` in the cardinal direction and the size of the :ref:`exhale_class_classAABB`. The generated file is saved to Minecraft/run/procedural.json.

class tomcat::model::MessageNode¶

This class represents a node in a factor graph used for exact inference with message passing algorithms.

struct MessageContainer¶

This struct stores a map between a node’s name (label + time step) and the message the emitted by this node. It’s used to store the incoming messages of a node where the key node names in this struct would indicate the nodes the messages came from.

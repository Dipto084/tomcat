class Pit : public AABB¶

This class represents a special AABB of type = “pit”. It is not hollow and is treated as having a roof.

Public Functions

Pit(int id, std::string material, Pos *topLeft, Pos *bottomRight)¶

Construct a new Pit object.

Parameters

  • id: The id to assign this Pit AABB

  • material: The base material the Pit is made of

  • topLeft: The coordinates of the top left of the AABB from the top view of the X-Z plane. Y coordinate should be lowest here.

  • bottomRight: The coordinates of the bottom right of the AABB from the top view of the X-Z plane. Y coordinate should be maximum here.

Destroy the Pit object.

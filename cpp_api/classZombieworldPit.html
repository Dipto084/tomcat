class ZombieworldPit : public AABB¶

This class represents a special AABB of type = “pit”. It is not hollow and is treated as having a roof.

Public Functions

ZombieworldPit(std::string id, Pos &topLeft, std::string fluid)¶

Construct a new Pit object.

Parameters

  • id: The id to assign this Pit AABB

  • topLeft: The coordinates of the top left of the AABB from the top view of the X-Z plane. Y coordinate should be lowest here.

  • fluid: The fluid that the pit is filled with. water or lava

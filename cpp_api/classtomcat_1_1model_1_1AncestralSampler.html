class tomcat::model::AncestralSampler : public tomcat::model::Sampler¶

Generates samples by ancestral sampling process. When data is provided for specific nodes, samples are not generated for these nodes and the samples generated for the latent nodes will be conditioned on their parent node’s data if provided.

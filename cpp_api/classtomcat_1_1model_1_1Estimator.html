class tomcat::model::Estimator¶

Represents a generic estimator for a DBN model.

Subclassed by tomcat::model::SumProductEstimator, tomcat::model::TrainingFrequencyEstimator

Estimator(std::shared_ptr<DynamicBayesNet> model, int inference_horizon, const std::string &node_label, const Eigen::VectorXd &assignment = Eigen::VectorXd(0))¶

Creates an abstract estimator.

Parameters

  • model: DBN

  • inference_horizon: how many time steps in the future estimations are going to be computed for

  • node_label: label of the node estimates are going to be computed for

  • assignment: fixed assignment (for instance, estimates = probability that the node assumes a value x, where x is the fixed assignment). This parameter is optional when the inference horizon is 0, but mandatory otherwise.

class AABB¶

This file defines the members and methods implemented as part of the AABB class.

This class represents an Axis Aligned Bounding Box as seen from the top-view of the Minecraft X-Z plane.

Pos getRandomPosAtBase(int, int, int, int)¶

Gets a random position in the AABB such that the y coordinate of the returned value is set to the top left y value which is considered the base.

Return

Pos

Parameters

  • offsetPosX: How far away from the left wall should the position be. Defaults to 1

  • offsetNegX: How far away from the right wall should the position be. Defaults to 1

  • offsetPosZ: How far away from the bottom wall should the position be. Defaults to 1

  • offsetNegZ: How far away from the top wall should the position be. Defaults to 1

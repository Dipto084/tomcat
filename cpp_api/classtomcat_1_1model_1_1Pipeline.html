class tomcat::model::Pipeline¶

This class represents a pipeline where a model can be trained and tested according to some training procedure and evaluation metrics. A pipeline can also be used to publish estimates from a pre-trained model on the fly as data comes in real time. This can be achieved by using an online estimation process in the pipeline. This feature is one of the cores of the ToMCAT’s communication system as other modules can keep track of ToMCAT’s calculations on the fly.

void execute()¶

Executes the pipeline. This operation splits the data set, trains a model, saves it (optional), estimate values over the trained model given unseen data, aggregate results (optional) and writes the outcomes of the execution to an output stream.

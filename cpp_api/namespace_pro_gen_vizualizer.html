Detailed Description¶

Filename: pro_gen_vizualizer.py Author: Adi Banerjee Purpose: This program will vizualize the procedurally generated map from the low_level_map.json file.

It does not accept alternate filenames. The vizualized map is saved in the same directory as script with the name “vizualized_plt.png”

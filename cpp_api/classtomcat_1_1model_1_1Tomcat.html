Generates synthetic data for the instantiated model.

Parameters

  • random_generator: random number generator

  • num_samples: number of samples to generate

  • output_folder: folder where the samples must be saved

  • equal_until: max time step for equal samples. After this time step, samples are not required to be the same.

  • max_time_step: generate data up to this time step.

  • excluding: labels of nodes which samples must not be saved

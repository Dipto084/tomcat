Computes the product of all the incoming messages (except the one coming from the target node) and the potential function. Next, marginalizes out the incoming nodes and returns the resultant message.

Return

Message

Parameters

  • template_target_node: template instance of the node where the message should go to

  • template_time_step: time step of this node where to get the incoming messages from. If the template node belongs to the repeatable structure, this information is needed to know which time step to address to retrieve the incoming messages.

  • target_time_step: real time step of the target node

  • direction: direction of the message passing

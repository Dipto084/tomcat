File gridworld_generator.cpp¶

↰ Parent directory (src/cpp/pro_gen)

Defines an algorithm and script to generate the JSON representing an N x N gridworld of cubic Class AABB with options to specify the separation between Class AABB in the cardinal direction and the size of the Class AABB. The generated file is saved to Minecraft/run/procedural.json.

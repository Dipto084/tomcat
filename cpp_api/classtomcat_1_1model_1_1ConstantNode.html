std::string get_timed_name() const override¶

Returns the node’s unique id in an unrolled DBN. This id is a combination of the node’s label and the time step where the timed instance node was placed in the unrolled DBN. If the node was not added to a DBN or this was not yet unrolled, the time step of the node is its default value.

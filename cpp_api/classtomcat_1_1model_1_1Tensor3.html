Tensor3 coeff_wise_and(int axis) const¶

Returns the bitwise-and of all the coefficients in a tensor in a given axis. All numbers greater greater 0 are considered as true for the purpose of this logical operation. Negative numbers are neutral and preserved in the final tensor. This operation shrinks the number of coefficients in the axis where the operation is performed but it preserves the number of axes.

Return

Mean.

Parameters

  • axis: axis where the bitwise-and must be computed along

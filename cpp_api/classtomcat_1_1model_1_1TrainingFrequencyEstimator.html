class tomcat::model::TrainingFrequencyEstimator : public tomcat::model::Estimator¶

This estimator is based on the relative frequencies of the observations over the training data for each time step on an unrolled DBN. For instance, the probability of observing a value x in time step t for a given node will be proportional to the number of times the value x is observed in the training data for that node at time t.

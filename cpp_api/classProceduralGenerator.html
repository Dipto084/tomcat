class ProceduralGenerator¶

This class represents a Procedural Generator which is meant to serve as a library of general methods that can be used for specific map generating algorithms.

Public Functions

Block getRandomVictim(Pos*, double, boost::random::mt19937*)¶

Uses the boost random mt19937 object to randomly choose a green or yellow victim and returns the appropriate block (prismarine or gold)

Return

Block The victim block

Parameters

  • pos: The position at which the victim must be placed

  • greenBias: Value between 0 and 1 for how probable it should be for the victim to be green

  • gen: Object used in random integer distribution for finding a random integer

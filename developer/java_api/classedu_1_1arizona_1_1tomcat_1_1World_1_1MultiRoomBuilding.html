Detailed Description

This file contains the class MultiRoomBuilding which extends the class Building. The instances of this class are initialized with x,y,z coordinates for the main room of the building.

At the time of creation the MultiRoomBuilding object only has the one main room. Additional rooms can be added through the addRoom method. The additional rooms added in this manner are called additional rooms and are separate from the main room the object is initialized with.

void edu.arizona.tomcat.World.MultiRoomBuilding.markRoomAsFilled ( BlockPos  coordinate)
inline

When given a set of coordinates as an integer array x,y,z, this method will mark the room of the building with that coordinate as filled. If the coordinate passed is that of the main room, then that room will be marked as filled.

If the coordinates passed are not of the main room or any of the additional rooms, then nothing will change.

coordinate- BlockPos object of coordinates
